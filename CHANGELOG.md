# Changelog

All notable changes appear here.

## 0.1.0 - 2026-07-23

### Added

- Source-first SkillsMP search and comparison workflow.
- Explicit standard and heavy modes.
- Dependency-free Node.js search runtime for Windows, macOS, and Linux.
- PowerShell compatibility wrapper for the original command style.
- Content scoring based on task fit, useful method delta, current practice, actionability, and validation.
- Project-scoped installation guidance through the cross-platform `skills` CLI.
- Mocked tests that consume no SkillsMP quota.
- Friendly README, examples, contribution guidance, and security policy.
